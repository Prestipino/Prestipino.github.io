EXAFS Resources and Spectra Databases

This page gathers a selection of EXAFS (Extended X-ray Absorption Fine Structure) databases and reference resources. These links provide access to experimental spectra, theoretical data, and reference materials for X-ray absorption spectroscopy (XAS) studies. They include both academic repositories and open-access initiatives supporting data sharing within the XAS community.

online tools

  • XrayDB — Python-based X-ray absorption database (edge energies, scattering factors, etc.).
  • Atomic Scattering Factors (UW) — Periodic table view for atomic scattering data.

Databases

  • Canadian Light Source – XAS Database — XAS and EXAFS spectra from the CLS synchrotron.
  • MDR XAFS DB — Comprehensive XAFS spectra collection hosted by Hokkaido University.
    • International XAFS DB Portal — Comprehensive XAFS spectra collection hosted where.
  • XASlib Element Browser — Interactive interface for browsing XAS data by element.
  • SSRL Materials EXAFS Spectra — Reference spectra from the Stanford Synchrotron Radiation Lightsource.
  • SSHADE / FAME Database — European database for solid-state spectroscopy and XAS data.
  • ACXAS Database (HZDR) — Database for Actinide and Complex XAS spectra.
  • NIMS MDR XAFS Collection — Japanese materials data repository hosting EXAFS datasets.
  • Review: Open XAS Data Initiatives (2023) — Recent review on data sharing and interoperability in XAS.

These links provide valuable data and tools for researchers working on X-ray absorption spectroscopy, from EXAFS fitting and interpretation to benchmarking and machine learning applications.
