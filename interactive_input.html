FullProf Files and Tools

This section provides several input files (.pcr) for the FullProf software suite, used for structure refinement from X-ray or neutron diffraction data. Each link opens an interactive page containing the original input file, enriched with explanatory comments and tooltips that describe the meaning and role of individual PCR parameters.

These examples illustrate a range of experimental and methodological configurations, from classical synchrotron data to recent multipattern refinement approaches. A complementary link is also provided for Faults, another program within the FullProf suite, dedicated to the modeling of structural defects and stacking disorder.

  • MOF.pcr — Classic format, synchrotron data.
  • multiP.pcr — New multipattern format (D8 diffractometer).
  • GEM.pcr — Multipattern neutron data from GEM.
  • LiNiO2.flts — FullProf suite program for modeling structural defects.

These resources are intended to support the understanding and practical use of the FullProf suite in both research and teaching contexts, by providing commented examples and interactive guidance.
